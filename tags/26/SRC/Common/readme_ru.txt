Отличия от оригинального 7-ZipFar
---------------------------------
 * Поддержка Far 2 и Far x64.
 * Пытается открыть только те файлы, которые соответствуют списку файловых масок.
 * Сохраняет режим панели и порядок сортировки.
 * При распаковке выделяет место под файл с целью уменьшения фрагментации.
 * Позволяет отключить ненужные форматы архивов (PE к примеру).
 * Возможность выбора формата архива вручную.
 * Некоторые другие мелкие улучшения.


Как это использовать?
---------------------
Установить плагин в FAR'е. (Скопировать файлы в папку Plugins\7-Zip)
В ту же папку (или куда-нибудь в PATH) сложить 7z.dll из дистрибутива 7-Zip.
Настроить параметры плагина по желанию.
Сам архиватор 7-Zip фактически не требуется.
